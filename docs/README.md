# HowsHous Architecture & Design Documentation

This directory contains comprehensive diagrams and documentation for the HowsHous smart boarding house solution.

## Contents

- **01-system-architecture.puml** - Overall system architecture (PlantUML)
- **02-use-case-diagram.puml** - Use case diagram by user role
- **03-class-diagram.puml** - Core data models and relationships
- **04-sequence-diagrams.puml** - Key user flows and interactions
- **05-package-diagram.puml** - Module organization
- **06-deployment-diagram.puml** - Infrastructure setup
- **07-database-schema.puml** - Firestore collections structure
- **08-user-workflows/** - Detailed workflows for each user type

## How to View These Diagrams

These files use PlantUML syntax. You can render them using:
- [PlantUML Online Editor](http://plantuml.com/plantuml/uml/)
- VS Code extensions (PlantUML)
- GitHub (PlantUML renders automatically in many markdown contexts)

## Project Overview

**HowsHous** is a smart boarding house solution for Baguio City that connects tenants with landlords through a comprehensive platform featuring:

- Listing management and search
- Real-time messaging system
- Contract management
- AI-powered assistance
- Analytics and metrics
- User verification and ban management
- Issue/maintenance tracking
